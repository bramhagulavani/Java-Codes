## 🛠️ How to Run the Code

### Prerequisites
- **Java JDK 11+** installed — [Download here](https://www.oracle.com/java/technologies/downloads/)
- Any IDE: **IntelliJ IDEA**, **Eclipse**, **VS Code** (with Java Extension Pack)

### Run a file locally

```bash
# 1. Clone the repository
git clone https://github.com/bramhagulavani/Java-Codes.git

# 2. Navigate to the folder
cd Java-Codes

# 3. Compile a Java file
javac FirstProgram.java

# 4. Run it
java FirstProgram
```
