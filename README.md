# Java Codes

This repository is for Java practice. It contains small programs, basic problem solving, OOP examples, DSA practice, LeetCode solutions, and college assignments.

## What is inside

- `Basics/` - simple Java programs like patterns, factorial, Fibonacci, prime checks, and number problems
- `College/` - class assignments and lab work
- `DSA/` - data structures and algorithm practice
- `LeetCode/` - LeetCode solutions by difficulty
- `OOP_Chap/` - OOP practice for abstraction, encapsulation, inheritance, and polymorphism

## Purpose

The main goal of this project is to improve Java skills through consistent practice. The code here is meant for learning, revision, and tracking progress over time.

## How to run

Make sure Java is installed, then compile and run any file from the terminal.

```bash
javac Basics/FirstProgram.java
java -cp Basics FirstProgram
```

If a file does not use a package, run it from the folder where the file is located:

```bash
cd Basics
javac FirstProgram.java
java FirstProgram
```

## Notes

- New practice programs will be added over time.
- Folder names may grow as more topics are covered.
